# EnglishCard - приложение для изучения английского языка

## Описание проекта

EnglishCard - это веб-приложение для изучения английских слов. Приложение позволяет пользователям учить слова, добавлять новые слова в свой словарь, удалять выученные слова и отслеживать статистику прогресса.

Проект разработан в рамках курсовой работы по курсу "Базы данных".

## Функциональность

- Автоматическая регистрация пользователя
- Изучение слов в формате викторины с 4 вариантами ответа
- Добавление новых слов в персональный словарь
- Удаление слов из персонального словаря
- Статистика правильных ответов и точности
- Просмотр схемы базы данных прямо в приложении

## Технологии

- Python 3.12+
- Streamlit - веб-фреймворк
- PostgreSQL - база данных
- psycopg2 - драйвер для PostgreSQL
- pandas - работа с данными

## Структура базы данных

База данных состоит из 4 таблиц:

| Таблица | Описание |
|---------|----------|
| users | Пользователи |
| words | Словарь (английские слова и переводы) |
| user_words | Связь пользователей со словами |
| statistics | Статистика правильных ответов |
